# Programmers-BE-Study
**Programmers-BE-Study**에 오신 것을 환영합니다! 🎉

우리는 프로그래머스 백엔드 데브코스 1기 전공자들이 모여 지식을 나누고 기술을 향상시키기 위해 만든 팀스터디입니다.

## 📚 우리의 목표
우리의 목표는 다음과 같습니다:
- 학습하면서 생긴 질문, 경험 공유하기.
- 백엔드 기술과 모범 사례를 학습하고 구현하기.
- 실제 프로젝트와 문제 해결에 참여하기.
- 커뮤니티와 인사이트 및 자료를 공유하기.
